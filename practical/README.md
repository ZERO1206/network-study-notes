# 实战方向 —— 实习必备

## 使用方式

每篇笔记是**指导手册**，不是演示。你需要自己打开终端执行命令，自己阅读输出，自己思考结果。

正确的学习节奏：
1. 读"任务"——你要做什么
2. 自己敲命令——不用复制粘贴
3. 看输出——你看到了什么？符合预期吗？
4. 回答"自测题"——用自己的话解释，点击折叠区对答案

## 需准备的环境

确保以下命令都存在：
```bash
which g++ ss curl nc tcpdump strace
```
